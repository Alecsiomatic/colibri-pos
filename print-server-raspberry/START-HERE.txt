  PASO 1: Flashear USB (15 minutos)
  ├─ Descarga: https://www.raspberrypi.com/software/
  ├─ Ejecuta: tools/flash-usb-windows.bat (sigue instrucciones)
  └─ Resultado: USB Kingston con Raspberry Pi OS

  PASO 2: Encender Raspberry Pi (5 minutos)
  ├─ Inserta USB en la Raspberry Pi
  ├─ Conecta cable ethernet (RJ45)
  ├─ Conecta corriente (arranca automáticamente)
  └─ Resultado: Raspberry Pi encendida y en red

  PASO 3: Instalar servidor (10 minutos)
  ├─ Desde PowerShell: ssh pi@printserver.local
  ├─ Password: supernovaprint
  ├─ Copia archivos: scp -r server pi@[IP]:~/
  ├─ Ejecuta: ./setup-raspberry.sh
  └─ Resultado: ¡Servidor funcionando! 🎉

┌────────────────────────────────────────────────────────────────────────────┐
│                      ⚡ INICIO RÁPIDO (SOLO 3 COMANDOS)                   │
└────────────────────────────────────────────────────────────────────────────┘

  Si ya tienes la Raspberry Pi con Raspberry Pi OS:

  # 1. Copiar archivos
  scp -r server pi@192.168.100.50:~/
  scp -r config pi@192.168.100.50:~/
  scp tools/setup-raspberry.sh pi@192.168.100.50:~/

  # 2. Conectar por SSH
  ssh pi@192.168.100.50

  # 3. Ejecutar instalador
  chmod +x setup-raspberry.sh && ./setup-raspberry.sh

  ¡LISTO! En 5 minutos tendrás el servidor funcionando. 🚀
